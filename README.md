# 🛡️ Campus Watchdog

## 📌 Project Overview

Campus Watchdog is an AI-powered campus security system designed to enhance safety through real-time monitoring and intelligent anomaly detection. The system uses computer vision techniques to analyze live camera feeds and identify suspicious or emergency situations automatically. It is built to assist campus authorities by reducing manual surveillance effort and enabling faster response to critical events.

## 🚀 Features

The system is capable of detecting three major anomalies from live video feeds. It identifies unusual crowd gathering in specific areas, detects unauthorized entry into restricted zones, and recognizes distress gestures using human pose estimation techniques. These features work together to provide real-time alerts and improve overall campus security.

## 🛠️ Tech Stack

The project is developed using Python as the core programming language. It utilizes YOLOv8 for object detection and OpenCV for real-time video processing. The backend is built using Flask, while the frontend interface is created using HTML, CSS, and JavaScript to provide a simple and interactive user experience.

## ⚙️ How to Run

To run the project, first clone the repository to your local system and navigate into the project folder. Then install all required dependencies using pip. After installation, run the main application file using Python. Once the server starts, open your browser and access the application through the local host address to view the live monitoring system.
